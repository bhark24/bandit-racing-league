Key Capabilities

Pit Road Warning System

Monitors live pit lane speed limit parameters, providing dynamic visual gauges and audio voice warnings to completely eliminate speeding penalties.

iRacing Live SDK Link

Under-the-hood real-time telemetry link that runs silently in the background with near-zero system resource usage, keeping your frame rates optimized.

VR Overlay Manager

Enables fully customized VR telemetry cards and pop-out modules, so you can place relative timetables and status boxes anywhere in your 3D cockpit.

App Setup Instructions

1. Verify & Download

Unlock the portal above using your official iRacing Customer ID and download the desktop installer (`Geezer_Dashboard_Setup.exe`). Run the setup wizard to install the software.

2. Launch & Link with iRacing

Open the Geezer Dashboard and make sure iRacing is running. The app will automatically establish a low-latency connection via the official iRacing SDK.

3. Telemetry ID Verification

Upon starting, the app verifies your active iRacing account number against the BRL registry. Telemetry records, incident thresholds, and driver profile settings will sync instantly.
